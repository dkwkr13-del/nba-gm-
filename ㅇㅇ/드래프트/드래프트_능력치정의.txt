§1. [입문능력치] 슬롯 정의 (Entry Attribute Slot Mapping)
────────────────────────────────────────────────────────────────────────────────

[입문능력치] 형식:
  FIN:a/b/c/d/e/f | SHT:a/b/c | PLY:a/b/c/d | DEF:a/b/c/d/e/f | ATH:a/b/c/d/e/f | IQ:a/b/c

각 카테고리의 슬롯 순서 (좌→우):

▶ FIN (피니셔 / 6슬롯)
  슬롯1: Close Shot       — 페인트 내 정지 슛 (포스트페이드 포함)
  슬롯2: Driving Layup    — 드라이브 후 레이업
  슬롯3: Driving Dunk     — 드라이브 후 덩크
  슬롯4: Standing Dunk    — 정지 상태 덩크 (신체+힘 의존)
  슬롯5: Post Hook        — 포스트업 훅 슛
  슬롯6: Post Control     — 포스트업 전반 (페이드어웨이 포함)

▶ SHT (슈팅 / 3슬롯)
  슬롯1: Mid-Range Shot   — 중거리 점프슛 (페인트 외곽~3점 라인 이내)
  슬롯2: Three-Point Shot — 3점 슛
  슬롯3: Free Throw       — 자유투

▶ PLY (플레이메이킹 / 4슬롯)
  슬롯1: Pass Accuracy    — 패스 정확도 및 창의성
  슬롯2: Ball Handle      — 볼 핸들링 (드리블 능력)
  슬롯3: Speed w/ Ball    — 볼 소지 시 스피드 (드리블 돌파력)
  슬롯4: Off-Ball Movement— 오프볼 움직임 (스크린 이용, 컷 타이밍)

▶ DEF (수비 / 6슬롯)
  슬롯1: Perimeter Defense— 외곽 수비 (온볼 압박, 컨테스트)
  슬롯2: Lateral Speed    — 수비 측면 이동속도 (풋워크)
  슬롯3: Defensive IQ     — 수비 읽기 (도움수비 타이밍, 스틸 판단)
  슬롯4: Interior Defense — 인사이드 수비 (페인트 저지, 포스트 수비)
  슬롯5: Block            — 블록샷 능력
  슬롯6: Steal            — 스틸 능력

▶ ATH (신체능력 / 6슬롯)
  슬롯1: Speed            — 직선 스피드 (코트 이동속도)
  슬롯2: Acceleration     — 초기 가속력 (퍼스트스텝)
  슬롯3: Vertical         — 수직 점프 (리바운드, 블록, 덩크 연동)
  슬롯4: Strength         — 체력·힘 (포스트 밀기, 몸싸움)
  슬롯5: Stamina          — 체력 지속력 (경기 후반 퍼포먼스 유지)
  슬롯6: Hustle           — 허슬 (루즈볼, 챔피언십 다이브, 화력 기여)

▶ IQ (농구 지능 / 3슬롯)
  슬롯1: Basketball IQ    — 전반적 경기 이해도 (공수 모두)
  슬롯2: Shot IQ          — 슛 선택 판단력 (양질의 슛 선택)
  슬롯3: Clutch           — 클러치 상황 능력치 보정 (경기 마지막 2분)

예시 읽는 법:
  FIN:72/78/65/28/48/52
  → Close Shot 72 / Driving Layup 78 / Driving Dunk 65
    Standing Dunk 28 / Post Hook 48 / Post Control 52

  DEF:58/78/73/36/40/58
  → Perimeter Defense 58 / Lateral Speed 78 / Defensive IQ 73
    Interior Defense 36 / Block 40 / Steal 58

§2. [성향] 코드 정의 (Tendency Code Definitions)
────────────────────────────────────────────────────────────────────────────────

[성향] 형식:
  슛선택: 코드1/코드2/… | 플레이: 코드1/코드2/… | 수비: 코드1/코드2/… | 리바: 코드1/코드2

총 17개 코드 — 각 선수는 아키타입에 맞는 코드 조합 보유

▶ 슛 선택 (Shot Type Tendency) — 6종
  3PT  : 3점슛 선호 (오픈 시 3점 먼저 선택)
  MID  : 중거리슛 선호 (미드레인지 활용)
  DRV  : 드라이브 선호 (페인트 침투 우선)
  PST  : 포스트업 선호 (인사이드 공격 시작)
  PUL  : 풀업 점퍼 선호 (드리블 정지 후 슛)
  C&S  : 캐치앤슛 선호 (패스 받자마자 슛)

▶ 플레이 스타일 (Play Style Tendency) — 5종
  PAS  : 패스 우선 (득점보다 동료 활용)
  ISO  : 아이솔레이션 선호 (1대1 상황 만들기)
  CUT  : 오프볼 컷 선호 (스크린 없이 시간차 컷)
  PNR  : 픽앤롤 볼핸들러 선호 (PNR 작전 주도)
  SCR  : 스크린 설정 선호 (팀 오펜스 동선 창출)

▶ 수비 성향 (Defensive Tendency) — 4종
  SA   : 스틸 시도 적극성 (인터셉트 도전 빈도)
  BA   : 블록 시도 적극성 (샷 컨테스트 빈도)
  PPR  : 온볼 압박 성향 (상대 볼핸들러 추격)
  HLP  : 도움수비 성향 (로테이션, 포기수비 실행)

▶ 리바운드 (Rebound Tendency) — 2종
  CRB  : 공격 리바운드 적극성 (오펜시브 리바 추구)
  BOX  : 박스아웃 우선 성향 (수비 리바 시 위치 선점)

예시:
  [성향] 슛선택:3PT/DRV/C&S | 플레이:PNR/CUT | 수비:BA/HLP | 리바:BOX
  → 3점+드라이브+캐치앤슛 선호 / 픽앤롤+컷 플레이 / 블록+도움수비 / 박스아웃 중시

§3. 컴바인 측정치 → [입문능력치] 슬롯 매핑 (Combine → Attribute Mapping)
────────────────────────────────────────────────────────────────────────────────

컴바인 결과는 [입문능력치] 슬롯에 직접 가산/감산된다.
(출처: 드래프트_컴바인시스템.txt SECTION 11 / 룰_드래프트.txt §91-B)

▶ 수직 점프 (도움닫기 기준)
  40인치+    : ATH슬롯3(Vertical) +5 / ATH슬롯1(Speed) +2
  35~39인치  : ATH슬롯3(Vertical) +3
  30~34인치  : 변동 없음
  30인치 미만: ATH슬롯3(Vertical) -3 / FIN슬롯3(Driving Dunk) -2

▶ 윙스팬/신장 비율
  1.10+      : DEF슬롯3(Defensive IQ) +3 / DEF슬롯5(Block) +2 / DEF슬롯6(Steal) +2
  1.05~1.09  : DEF슬롯3(Defensive IQ) +1
  1.00 미만  : DEF슬롯3(Defensive IQ) -2 / DEF슬롯5(Block) -1

▶ 레인 어질리티 (Lane Agility)
  10.5초 미만: DEF슬롯2(Lateral Speed) +3 / ATH슬롯1(Speed) +3
  10.5~11.0초: DEF슬롯2(Lateral Speed) +1
  11.5초 초과: DEF슬롯2(Lateral Speed) -2 / ATH슬롯2(Acceleration) -1

▶ 3/4 코트 스프린트
  3.0초 미만 : ATH슬롯1(Speed) +4 / ATH슬롯2(Acceleration) +2
  3.0~3.3초  : ATH슬롯1(Speed) +2
  3.6초 초과 : ATH슬롯1(Speed) -2

▶ 벤치프레스 횟수 (185lb / 84kg)
  15회+      : ATH슬롯4(Strength) +4
  10~14회    : ATH슬롯4(Strength) +2
  5회 미만   : ATH슬롯4(Strength) -2

▶ 슈팅 드릴 — 캐치앤슛 성공률
  80%+       : SHT슬롯2(Three-Point Shot) +5
  70~79%     : SHT슬롯2(Three-Point Shot) +2
  60% 미만   : SHT슬롯2(Three-Point Shot) -3

▶ 자유투 드릴 성공률
  90%+       : SHT슬롯3(Free Throw) +4
  80~89%     : SHT슬롯3(Free Throw) +2
  70% 미만   : SHT슬롯3(Free Throw) -2

▶ 농구 IQ 테스트 점수
  90+        : IQ슬롯1(Basketball IQ) +5 / IQ슬롯2(Shot IQ) +3 / DEF슬롯3(Defensive IQ) +3
  80~89      : IQ슬롯1(Basketball IQ) +2
  70 미만    : IQ슬롯1(Basketball IQ) -3

▶ HARD LOCK 원칙 (컴바인 결과가 절대적 기준 설정)
  - 수직점프 40인치+ 선수의 ATH슬롯3(Vertical)은 최소 75 보장
  - 레인 어질리티 10.5초 미만 선수의 DEF슬롯2(Lateral Speed)는 최소 72 보장
  - 벤치프레스 0회 선수의 ATH슬롯4(Strength)는 최대 55 제한
  - 슈팅 드릴 60% 미만 선수의 SHT슬롯2(Three-Point Shot)는 최대 60 제한

§6. [입문능력치] OVR 산정 공식 (OVR Calculation)
────────────────────────────────────────────────────────────────────────────────

OVR = 포지션별 가중치 적용 평균

공통 기준 가중치 (포지션별 세부 조정):
  FIN 카테고리 평균 × 0.20
  SHT 카테고리 평균 × 0.18
  PLY 카테고리 평균 × 0.17
  DEF 카테고리 평균 × 0.20
  ATH 카테고리 평균 × 0.15
  IQ  카테고리 평균 × 0.10

포지션별 보정:
  PG/SG:  PLY 가중치 +0.05 / FIN 가중치 -0.03
  SF:     균형 (기준값 유지)
  PF:     FIN 가중치 +0.03 / DEF 가중치 +0.02 / PLY 가중치 -0.05
  C:      FIN 가중치 +0.05 / DEF 가중치 +0.05 / SHT 가중치 -0.07
