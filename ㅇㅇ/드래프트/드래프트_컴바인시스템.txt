▶ SECTION 0 : 컴바인 일정 & 참가 자격 (Schedule & Eligibility)
────────────────────────────────────────────────────────────────────────────────
■ 개최 일정 (매년 5월 중순, 시카고 / 실제 NBA 컴바인 동일)
  Day 1 (측정일):  신체 측정 전 항목 + 메디컬 스크리닝
  Day 2 (운동일):  운동 능력 테스트 전 항목 (버티컬 / 어질리티 / 스프린트 / 벤치)
  Day 3 (스킬일):  스킬 드릴 4종 (슈팅 / 핸들링 / P&R / 패스)
  Day 4 (게임일):  5대5 스크리미지 풀 게임
  Day 5 (면담일):  팀 인터뷰 (구단별 15분 × 최대 30팀)

■ 참가 자격
  ① 드래프트 신청 선수 전원에게 컴바인 초청장 발송 (기본)
  ② 상위 투영 선수 (Big Board Top 20 이내) → 출전 거부 옵션 보유 (10% 확률 발동)
  ③ 해외 리그 소속 선수 (유럽·기타) → 일부 선택적 참가 허용
     - 신체 측정 + 메디컬만 참가 시 → 운동/스킬 항목 "N/A"
     - 전 항목 참가 시 동일 기준 적용
  ④ 부상 중인 선수 → 해당 항목 "부상 면제" 처리 (메디컬 기록 별도 관리)

■ 항목별 참가 여부 영향
  전 항목 참가 (5일) → 컴바인 종합 점수 정상 산출
  Day 1-2만 참가   → 신체·운동 항목만 반영, 스킬·IQ 평균 B등급으로 대체
  Day 1만 참가     → 신체 측정값만 실측, 나머지 전부 "스카우팅 추정치"
  불참             → 전 항목 "스카우팅 추정치" / HARD LOCK 미적용

▶ SECTION 1 : 신체 측정 (Anthropometric Measurements)
────────────────────────────────────────────────────────────────────────────────
측정 항목 (6종):
  [1] 신발 착용 신장 (Height w/ Shoes)        단위: cm
  [2] 신발 미착용 신장 (Height w/o Shoes)     단위: cm
  [3] 윙스팬 (Wingspan)                       단위: cm
  [4] 스탠딩 리치 (Standing Reach)            단위: cm
  [5] 체중 (Weight)                           단위: kg
  [6] 핸드 사이즈 (Hand Size)                 단위: cm (길이/폭)

평가 기준:
  윙스팬/신장 비율 ≥ 1.07 → "장완 등급" 보너스 (+3 수비 등급)
  스탠딩 리치 ≥ 275cm    → "림 프로텍터 잠재력" 태그 자동 부여
  핸드 사이즈 ≥ 24cm     → "볼 컨트롤 보너스" +2

■ 드래프트 파일 수치 갱신 규칙
  ※ 컴바인 실측 결과가 기존 드래프트 파일의 신체 수치와 다를 경우 자동 업데이트
  - 신장·윙스팬은 컴바인 실측값으로 덮어쓰기 (드래프트 파일 괄호 내 원래 값 보존 가능)
    예) 윙스팬: 211cm → "211cm (컴바인 실측: 213.5cm)"
  - 체중은 컴바인 당일 측정값 기록 (시즌 중 변동 반영 가능성)
  - 기존 드래프트 파일 ATH 수치 중 Vertical / Speed 항목은 SECTION 10 변환 로직 결과로
    입문 능력치 FLOOR가 재설정됨 (기존 값보다 낮을 경우 하향 보정 가능)

▶ SECTION 2 : 운동 능력 테스트 (Athletic Testing)
────────────────────────────────────────────────────────────────────────────────
테스트 항목 (5종):
  [1] 수직 점프 — 서서 (No-Step Vertical)         단위: cm
  [2] 수직 점프 — 도움닫기 (Max Vertical)         단위: cm
  [3] 레인 어질리티 (Lane Agility)                단위: 초 (낮을수록 우수)
  [4] 셔틀런 (3/4 Court Sprint)                   단위: 초 (낮을수록 우수)
  [5] 벤치프레스 (Bench Press — 185lbs/84kg)      단위: 횟수

등급 기준표:
┌─────────────────┬────────────┬────────────┬────────────┬────────────┐

├─────────────────┼────────────┼────────────┼────────────┼────────────┤
│ Max Vertical     │ ≥ 104cm    │ 95~103cm   │ 80~94cm    │ < 80cm     │
│ Lane Agility     │ ≤ 10.5초   │ 10.6~11.0s │ 11.1~11.8s │ > 11.8초   │
│ 3/4 Court Sprint │ ≤ 3.0초    │ 3.0~3.2초  │ 3.2~3.5초  │ > 3.5초    │
│ Bench Press      │ ≥ 15회     │ 10~14회    │ 5~9회      │ < 5회      │

▶ SECTION 3 : 스킬 드릴 (Skills Drills)
────────────────────────────────────────────────────────────────────────────────
드릴 항목 (4종):
  [1] 슈팅 드릴 (Shooting Drill)
      - 5개 스팟(코너×2, 윙×2, 탑) 각 3회 = 최대 15개
      - 결과 출력: X/15 (예: 11/15 → "슈팅 합격선")
      - 기준: 11/15 이상 → "신뢰할 수 있는 슈터" 태그

  [2] 볼핸들링 드릴 (Ball-Handling Circuit)
      - 좌/우 드리블, 체인지오브페이스, 풀코트 드리블
      - 등급: A (깔끔) / B (무난) / C (개선 필요)

  [3] 픽앤롤 공격/수비 드릴 (Pick-and-Roll Drill)
      - 공격자 / 수비자 양 역할 수행
      - IQ 점수 0~10 부여

  [4] 패스 정확도 드릴 (Passing Accuracy Drill)
      - 무빙 타겟 패스 10회 → X/10 결과
      - 기준: 8/10 이상 → "패스 마스터" 태그

▶ SECTION 4 : 5대5 게임 (5-on-5 Scrimmage)
────────────────────────────────────────────────────────────────────────────────
형식: 4쿼터 × 6분 = 24분 풀 게임 / 랜덤 팀 배정
평가 항목: 득점 / 리바운드 / 어시스트 / 스틸 / 블록 / 턴오버
추가 평가: 코치 점수 (0~10) — 전술 이해도, 커뮤니케이션, 허슬

스크리미지 결과 예시 출력 형식:
  [PLAYER] 12pts / 7reb / 3ast / 1stl / 1blk / 2to | 코치점수: 8.5/10
  → "컴바인 MVP" 태그 조건: 득점 1위 + 코치점수 8.0+

▶ SECTION 5 : 메디컬 & 인터뷰 (Medical & Interview)
────────────────────────────────────────────────────────────────────────────────
메디컬 검사:
  - X-레이 / MRI 통합 판정 → 결과: 클린 / 경미한 이슈 / 중대한 이슈
  - "레드 플래그" 발생 시: 해당 선수 순위 자동 5~15계단 하락
  - "클린" 판정 시: 해당 선수 순위 보정 없음 (기본값)

인터뷰 평가:
  - 구단 면담 수: 최대 30개 팀 × 15분 면담
  - IQ 테스트 점수 (0~100): 80+ → "High Basketball IQ" 태그
  - 멘탈 평가: 압박 상황 대응, 리더십, 팀 케미스트리 의지
  - 인터뷰 결과 등급: S (극호평) / A / B / C (우려)

▶ SECTION 6 : 컴바인 종합 등급 (Overall Combine Grade)
────────────────────────────────────────────────────────────────────────────────
종합 점수 산출 방식 (100점 만점):
  신체 측정    15점 (윙스팬 비율, 스탠딩 리치 등)
  운동 능력    25점 (버티컬, 어질리티, 스프린트 합산)
  스킬 드릴    20점 (슈팅 + 핸들링 + P&R + 패스)
  5대5 게임    25점 (스탯 + 코치 점수 복합)
  메디컬       10점 (클린=10 / 경미=5 / 중대=0)
  인터뷰        5점 (S=5 / A=4 / B=3 / C=1)

컴바인 등급 테이블:
  90~100점 → ★★★★★ "컴바인 킹"     → Big Board +3~5 상승
  80~89점  → ★★★★☆ "컴바인 엘리트" → Big Board +1~2 상승
  65~79점  → ★★★☆☆ "컴바인 통과"   → Big Board 변동 없음
  50~64점  → ★★☆☆☆ "컴바인 우려"   → Big Board -1~3 하락
  49점 이하 → ★☆☆☆☆ "컴바인 폭탄"  → Big Board -5~10 하락

▶ SECTION 9 : 팀별 Private Workout 시스템 (Private Workouts)
────────────────────────────────────────────────────────────────────────────────
※ 컴바인 종료(5월 말) ~ 드래프트 당일(6월) 사이 진행
※ 각 구단이 드래프트 대상 선수를 개별 초청해 자체 시설에서 진행하는 1:1 심층 평가

■ 기본 규칙
  - 각 구단 최대 8명까지 private workout 실시 가능 (드래프트 전까지)
  - 선수 1명은 최대 6개 팀의 workout 참가 가능 (체력·일정 관리 고려)
  - 워크아웃 결과는 해당 구단만 열람 가능 (비공개)
  - 컴바인 불참 선수도 private workout 참가 가능

■ 워크아웃 구성 (구단 자율 설계 — 기본 템플릿)
  ① 1:1 드릴 세션     (30분) — 구단 코치진 직접 지도
  ② 시스템 적합도 테스트 (20분) — 팀 전술 시뮬레이션
  ③ 심층 인터뷰       (30분) — 구단 GM/HC 직접 면담
  ④ 피지컬 재측정     (10분) — 신장·체중·윙스팬 재확인 (선택)

■ 워크아웃 결과 효과
  워크아웃 후 해당 구단의 해당 선수 드래프트 카드 → DEEP SCOUTED 등급 자동 해제
  추가 정보 공개:
    - 팀 전술 적합도 (상세 분석)
    - 코치진 심층 평가 (호감도 수치)
    - 루키 계약 의지 강도 (높음/보통/낮음)
    - 워크아웃 중 특기 발견 여부 (Hidden Trait 힌트 추가 1개)

■ 워크아웃 특수 이벤트
  [임팩트 워크아웃] — 워크아웃 선수 5% 확률
  : "워크아웃에서 압도적 퍼포먼스. 해당 구단 Big Board 내 +5 급등."

  [워크아웃 취소] — 상위 투영 선수 8% 확률
  : "에이전트 측이 특정 팀과의 워크아웃 거부. 해당 팀과의 관계 냉각."
    → 해당 팀 드래프트 시 해당 선수 선택 우선도 -30%

  [워크아웃 공개 유출] — 2% 확률
  : "비공개 워크아웃 내용이 SNS에 유출. 선수 시장 가치 변동."
    → 결과에 따라 Big Board ±3~8 변동 / 구단 내부 정보 유출 패널티

▶ SECTION 10 : 컴바인-드래프트 연동 규칙 (Combine-Draft Integration)
