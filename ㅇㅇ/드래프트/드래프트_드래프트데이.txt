▶ 1-2. 그린룸 (Green Room) 시스템
────────────────────────────────────────────────────────────────────────────────
  대상: 컴바인 기준 Big Board Top 20 선수 자동 초청
        (초청 거부 가능 — 일부 선수 자택 파티 선택)

  그린룸 체류 효과:
    ① 픽 지명 시 연단 입장 씬 자동 생성 (카메라 반응 포함)
    ② 기대 픽보다 늦게 지명 시 → "슬라이드 이벤트" 발동
    ③ 기대 픽보다 일찍 지명 시 → "서프라이즈 픽 이벤트" 발동

  슬라이드 이벤트 (기대 대비 -5픽 이상 하락):
    → 선수 멘탈 -5~10 / 해당 팀과 계약 협상 긴장도 상승
    → 언론 스토리 자동 생성: "[선수명]의 믿기 힘든 추락"
    → 주의: 슬라이드 선수는 신인 시즌 동기부여 보너스 +10 (반발 심리)

  서프라이즈 픽 이벤트 (기대 대비 +5픽 이상 상승):
    → 선수 멘탈 +10 / 지명 팀 팬 호응도 +15%
    → 언론 스토리: "[팀명]의 대담한 픽 — [선수명]의 잠재력은?"

  그린룸 불참 선수:
    → 자택/파티 지명 → 전화 통보 연출 / 연단 입장 씬 없음
    → 지명 팀 팬 현장 반응 없음 (SNS 반응으로 대체)

=== CHAPTER 2 : 픽 선택 시스템 ===

────────────────────────────────────────────────────────────────────────────────
▶ 2-1. 기본 선택 규칙
────────────────────────────────────────────────────────────────────────────────
  각 팀은 자신의 픽 순서가 되면 반드시 선택 행동을 수행해야 함:
    ① 선수 지명 (드래프트)
    ② 픽 트레이드 (다른 팀에게 픽 또는 선수와 교환)
    ③ 픽 패스 (불가 — NBA 규정상 반드시 지명 또는 트레이드 완료)

  픽 시간 초과 시:
    → 최초 경고 후 30초 추가 / 이후 리그 자동 최우선 지명 (Best Available)

────────────────────────────────────────────────────────────────────────────────
▶ 2-2. 트레이드 시스템 (Trade System)
────────────────────────────────────────────────────────────────────────────────
■ 드래프트 당일 트레이드 가능 자산
  - 현재 드래프트의 픽 (1·2라운드 모두)
  - 미래 드래프트 픽 (최대 2년 선행 — 단, 프로스펙트 룰 적용)
  - 기존 NBA 선수 (연봉 매칭 필요 — 드래프트 당일 예외 조항 있음)
  - 현금 (최대 $5M / 거래당)

■ 트레이드 성사 조건
  - 양 팀 GM 동의 필요
  - 리그 오피스 승인 (자동 처리 — 2분 내)
  - 연봉 총액 매칭: 트레이드 선수 연봉 합계 차이 ±25% 이내

■ 픽 가치 환산 기준표 (Trade Value Chart)
  ┌──────────┬───────────┬─────────────────────────────┐
  │ 픽 번호  │ 가치 포인트│ 비고                         │
  ├──────────┼───────────┼─────────────────────────────┤
  │ #1       │ 3000      │ 프랜차이즈 가치              │
  │ #2~3     │ 2200~2500 │ 슈퍼스타 후보                │
  │ #4~5     │ 1600~1900 │ 복권 최상위                  │
  │ #6~10    │ 900~1400  │ 확실한 로터리 가치           │
  │ #11~14   │ 500~800   │ 로터리 후반                  │
  │ #15~20   │ 250~450   │ 1라운드 후반                 │
  │ #21~30   │ 100~220   │ 후기 1라운드                 │
  │ #31~45   │ 40~90     │ 2라운드 상위                 │
  │ #46~60   │ 10~35     │ 2라운드 하위                 │
  └──────────┴───────────┴─────────────────────────────┘
  ※ 미래 픽은 현재 픽 가치의 70~90% 적용 (연도별 감가)
  ※ 두 팀 합의 시 차트 무시 가능 (단, 언론 평가 반영됨)

■ 드래프트 당일 트레이드 특수 규칙
  - 픽 발표 후 5분 이내까지만 해당 픽 트레이드 가능 (선수 발표 전)
  - "픽-포-픽 즉시 스왑": 양 팀 동시 카드 제시 방식 / 30초 내 확정
  - 트레이드 완료 시 리그 공식 발표 + 언론 자동 반응 생성

■ 프로스펙트 룰 (Prospect Rule)
  - 트레이드 가능한 미래 픽은 최대 2년 후까지
  - 동일 팀의 연속 2년 픽을 동시에 트레이드 금지
    (예: 2028 1라운드 + 2029 1라운드를 동시에 보내는 것 불가)
  - 예외: 현재 시즌 플레이오프 참가팀은 1년 선행 픽만 트레이드 가능

────────────────────────────────────────────────────────────────────────────────
▶ 2-3. 바이아웃 & 포기 픽 (Buyout / Pass Pick)
────────────────────────────────────────────────────────────────────────────────
  2라운드 픽 포기 조건:
    - 해당 팀의 로스터가 이미 가득 찬 경우 픽 포기 가능
    - 포기 시 리그에 픽 반환 → 해당 픽은 "미지명" 처리
    - 픽 포기 대신 현금 수령 가능 ($1M 상한)

  글로벌 드래프트 권리 (International Rights):
    - 지명 후 선수가 해외 잔류 결정 시 → 해당 팀 "드래프트 권리" 보유
    - 권리 보유 기간: 지명 시점으로부터 최대 3년
    - 3년 내 NBA 합류 불가 시 → 권리 소멸, 선수 다음 드래프트 재진입 가능
    - 권리 트레이드 가능 (기간 내, 픽 가치의 20~50% 수준)

▶ 3-3. 카드 공개 범위 정책 (Card Reveal Rules)
────────────────────────────────────────────────────────────────────────────────
1. 세부 능력치(0~99 개별 수치)는 드래프트 당일 절대 공개 불가
   → 1시즌 종료 후 오펜스 3종 부분 공개 시작
   → 3시즌 이상 경과 시 전체 공개 (통계 역산 가능 시점 기준)

2. 컴바인 측정값(cm / 초 / 횟수)은 전체 공개 (실제 NBA Draft Combine 정책 반영)

3. 아키타입·Ceiling/Floor 등급(문자 등급만)은 전체 공개

4. 능력치 범위 추정 등급(S/A+/A...)은 컴바인시스템 SECTION 11 변환 로직 자동 적용
   → 정확한 수치 아님 / ±1등급 오차 허용

5. 컴바인 불참 선수

    ① 전체 60명 지명 선수 → 해당 팀 로스터에 자동 추가 (Unsigned 상태)
    ② 계약 협상 단계 시작 (7일 이내 루키 스케일 계약 체결 원칙)
    ③ 해외 잔류 선택 선수 → 국제 선수 드래프트 권리로 분류 / 로스터 비포함
    ④ 미지명 선수 (61위 이하) → 자동으로 UDFA 계약 가능 상태
    ⑤ Big Board 최종 업데이트 + 드래프트 결과 아카이브 저장

────────────────────────────────────────────────────────────────────────────────
▶ 7-2. 미지명 자유계약 (UDFA — Undrafted Free Agent)
────────────────────────────────────────────────────────────────────────────────
  대상: 드래프트에서 지명받지 못한 선수 전원
  계약 형태: 2-way 계약 or 엑스hibition 계약 (비보장 최소)
  계약 기간: 드래프트 당일 자정~다음날 자정 (24시간 내 계약 러시)

  UDFA 경쟁 시스템:
    - 상위 UDFA (Big Board #61~70) → 여러 팀 동시 오퍼 가능
    - 계약 확정: 선수 에이전트 최종 선택 (연봉 외 출전 기회 중시)
    - 2-way 계약 (G리그 병행): NBA 연봉 일부 + G리그 연봉 분리

  UDFA 히든 잠재력:
    - UDFA 선수 중 5% → 2~3시즌 후 로스터 주요 선수로 성장
    - 해당 팀 GM 스카우팅 평판 대폭 상승 ("다이아몬드 발굴")

────────────────────────────────────────────────────────────────────────────────
▶ 7-3. 서머리그 연계 (Summer League Integration)
────────────────────────────────────────────────────────────────────────────────
  시기: 드래프트 후 약 3주 뒤 (7월 초, 라스베이거스 Summer League)
  대상: 루키 전원 + 2년차 이하 선수 + 팀 선택 베테랑 (최대 3명)

  서머리그 효과:
    ① 루키 초기 능력치 미세 조정 (5경기 평균 기준)
    ② 탁월한 퍼포먼스 → 기대치 상승 / 팬 관심도 증가
    ③ 부진한 퍼포먼스 → 적응 기간 연장 / 일부 팬 우려

  서머리그 특수 이벤트:
    [서머리그 MVP] — 해당 루키 팀 내 기대치 +10 / 미디어 노출 급증
    [부상 이탈]    — 2% 확률 / 정규 시즌 개막 전 회복 여부 판정
    [뜻밖의 발견]  — UDFA 선수가 서머리그 돌풍 / 정규 로스터 계약 전환 가능
