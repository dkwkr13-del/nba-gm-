### 10. 심리, 내러티브 및 사무국 (Drama & League Office)
* 50. [동적 라이벌리 (Dynamic Rivalry)]: 플레이오프 연속 매치업이나 벤치 클리어링 발생 시 새로운 라이벌 관계가 형성됩니다.
  - 라이벌 등급: Emerging / Heated / Historic (3단계)
  - Heated 이상: 해당 팀 간 경기 시 양 팀 선수 동기부여 +10%, 홈 팬 지지도 2배
  - Historic: 경기 전 내러티브 자동 출력, 관중 매진 효과
* 51. [계약해 버프 (Contract Year Boost)]: FA 직전 시즌 스탯 찌기(Stat Padding)로 득점 상승, 패스/수비 하락 현상 발생.
  - 발동 선수: 득점 +8%, USG% 자동 상승
  - 단점: 팀 케미스트리 -5, 수비 -5%
* 52. [좀비 본능 (Playoff Riser)]: 정규시즌엔 평범하다가 PO만 되면 능력치가 15% 상승하는 '강심장' 특성.
  - 발동 조건: 해당 특성 보유 선수가 플레이오프 진입
  - 반대 특성: 'Playoff Choker' — 정규시즌 활약하다 PO에서 10% 하락 (발동 확률 3%)
* 53. [심판의 원한 & 음모론 (Ref Grudge / Conspiracy)]: 특정 심판과 선수의 상성(Ref Grudge), 그리고 드래프트 로터리 조작 의혹(Frozen Envelope) 이벤트가 드물게 발생합니다.
  - Ref Grudge 등록 시: 해당 심판 배정 경기 파울 수 +2
  - Frozen Envelope: 빅마켓 팀이 저확률 1순위 당첨 시 발동 (리그 신뢰도 -5)
* 54. [SNS 및 팟캐스트 (Media Impact)]: 선수의 팟캐스트 발언이나 SNS 활동이 팀 케미스트리에 실시간으로 영향을 줍니다.
  - 긍정 발언: 팀 케미스트리 +5~+10
  - 부정·저격 발언: 팀 케미스트리 -10~-25, 리그 벌금 $25,000~$250,000
  - SNS 발언은 내러티브 이벤트 파일 [PD-07] 연동
* 55. [분노 버프 (Chip on Shoulder)]: 외부의 비판·무시를 받은 선수가 시즌 동안 동기부여 상승
  - 올스타 스나브: 해당 선수 시즌 나머지 기간 동기부여 +12%
  - ESPN 혹평 대상 팀: 5경기 한시적 +8%
  - 트레이드 요구 거절 당한 선수: 즉시 동기부여 -15% (분노) → 2주 후 +10% (분노 버프 전환)
* 56. [팀 케미스트리 수치 (Chemistry Score)]: 0~100 수치로 관리. 등급별 팀 효과·변동 요인·붕괴 이벤트 세부 규칙 → **섹션 49** 참조.

### 28. 내러티브 이벤트 파일 연동 규칙 (Narrative File Integration)

* 모든 이벤트 발동 조건과 효과는 **내러티브_이벤트.txt** 파일을 우선 참조합니다.
* 이벤트 발동 시 해당 파일의 [카테고리-번호] 코드로 조회합니다.
  - 예: `[PD-01]` → 선수 트레이드 요구 이벤트 내러티브 출력
* **유저 팀에 영향을 주는 이벤트** 발동 전 반드시 확인 메시지를 출력합니다:
  ```
  ⚠️ [{이벤트명}] 발동 조건 충족
  → {이벤트 요약 내용}
  → 진행하시겠습니까? [Y / N / 강제수정]
  ```
* **[강제 수정]** 명령어로 이벤트 결과 직접 지정 가능 (God Mode 적용)
* CPU 팀에서 발동된 이벤트는 자동 처리 후 뉴스 티커로 요약 보고합니다.

### 48. 사무국 특별 이벤트 확장 (League Office Special Events)

#### [리그 규정 변경 투표 시스템 (Rule Change Vote)]
매 시즌 종료 후 리그 사무국은 규정 변경안을 **투표**로 처리합니다.

**발동 조건:** 매 2시즌마다 변경 후보 1~3건 랜덤 제시 (30개 구단 투표, 18표 이상 가결)

| 변경안 | 찬성 영향 | 반대 영향 |
|---|---|---|
| **3점 라인 후퇴 (1피트)** | 슈팅 전문 팀 약화, 드라이브 팀 유리 | 3점 문화 유지 |
| **샷 클락 20초로 단축** | 속공 팀 유리, Pace 급상승 | 전술 팀 반발 |
| **플레이인 폐지** | 7~10시드 동기부여 감소 | 플레이오프 순수성 회복 |
| **6파울 → 7파울 아웃** | 수비 강도 완화, 파울 트러블 선수 기용 유연 | 수비 수준 하락 |
| **드래프트 로터리 균등화 (하위 4팀 동확률)** | 탱킹 억제 | 리빌딩 전략 타격 |
| **인시즌 토너먼트 폐지** | NBA Cup 없어짐 | 레귤러 시즌 흥행 감소 |
| **라인업 공개 의무화 (경기 1시간 전)** | 전술 분석 공정성 향상 | 감독 전략 자유도 감소 |
| **최대 계약 기간 4년으로 단축** | 선수 FA 자유도 증가 | 구단 장기 계획 타격 |

> 가결 시 **다음 시즌**부터 즉시 적용. 유저 팀은 투표 방향 선택 가능.

#### [노사분쟁 & CBA 재협상 (Labor Dispute & CBA Renegotiation)]
**발동 조건:** 매 6~8시즌마다 1회 CBA 만료 이벤트 발동

| 단계 | 내용 | 시뮬 영향 |
|---|---|---|
| **협상 시작** | 선수 노조 vs 구단주 연맹 협상 개시 | 6개월 전 예고 이벤트 |
| **협상 결렬 위기** | 30% 확률로 파업 위협 | FA 모라토리엄 연장, 루키 계약 동결 |
| **단기 타협 (1~2년 연장)** | 50% 확률 | 샐러리캡 소폭 조정, 큰 변화 없음 |
| **전면 합의 (새 CBA)** | 40% 확률 | 샐러리캡·택스·계약 규정 일부 변경 가능 |
| **실제 파업 (Lockout)** | 10% 확률 | 프리시즌 단축 or 정규시즌 경기 수 감소 (66~72경기) |

> 파업 발생 시: 샐러리캡 동결, 모든 계약 연기, 팬 지지도 -15% (전 팀)

### 49. 심리·모멘텀 심화 시스템 (Psychology & Momentum System)

#### [모멘텀 레벨 (Momentum Level)]
경기 중 팀·선수 단위 **모멘텀(0~5)**이 실시간 변동합니다.

| 모멘텀 수준 | 조건 | 팀 효과 |
|---|---|---|
| **Lv 0 (데드)** | 5점 차 이상 + 연속 실패 | 슛 성공률 -10%, 수비 집중도 -10% |
| **Lv 1 (보통)** | 기본값 | 보정 없음 |
| **Lv 2 (상승)** | 연속 3점 or 스틸+속공 성공 | 팀 효율 +5% |
